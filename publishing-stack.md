# Stack de publication recommandée

## Recommandation nette
**Astro + Markdown + déploiement simple**

## Pourquoi
- rapide
- SEO-friendly
- simple à maintenir
- parfait pour un site contenu-first
- faible complexité inutile

## Stack recommandée
- framework : Astro
- contenu : Markdown
- styles : minimal
- analytics : simple et sobre
- déploiement : Vercel ou Netlify

## Pourquoi pas plus complexe
Le projet n’a pas besoin d’un CMS lourd au départ. Il a besoin de contenu, structure, vitesse, et d’un workflow propre.

## Workflow initial
1. écrire en Markdown dans `projects/ai-blog/articles/`
2. publier ensuite dans le site statique
3. mettre à jour les contenus piliers régulièrement
4. créer les pages catégories ensuite
