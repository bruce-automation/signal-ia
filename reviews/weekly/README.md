# Weekly traffic and SEO review system

Purpose: run Signal IA with evidence instead of vibes.

## Cadence
- once per week
- same day if possible
- one markdown file per review in this folder

## Inputs to check
- Google Search Console performance report
- Google Search Console index coverage / pages report
- GoatCounter traffic dashboard
- published articles added during the week
- internal linking or credibility-page changes shipped during the week

## How to start a fresh review
```bash
npm run review:weekly
```

That command creates a dated review file from the template.

## Review rules
- keep it compact
- write numbers, not impressions
- end with 3 next actions maximum
- if there is no data yet, record that clearly instead of guessing
